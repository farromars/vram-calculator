## 八、已知问题与风险

| 风险项 | 影响 | 缓解方案 |
|--------|------|----------|
| 无自动化测试框架 | 所有测试依赖手工执行 | 后续引入 Playwright/Cypress |
| TypeScript检查被跳过 | 类型错误不会阻断构建 | 逐步启用 `ignoreBuildErrors: false` |
| ESLint检查被跳过 | 代码风格问题不会告警 | 逐步启用 `ignoreDuringBuilds: false` |
| 计算公式无单元测试 | 公式修改可能引入计算错误 | 后续为 `memory-formulas.ts` 编写Jest测试 |
| 性能估算为理论值 | TPS/TTFT与实际可能有偏差 | 标注"仅供参考"，后续校准 |
